## 📌 Project Overview

This project focuses on building a **machine learning model** to predict household appliance energy consumption based on environmental and temporal factors.

The goal is to:
- Improve **energy efficiency**
- Help users **reduce electricity costs**
- Support **smart grid management**

---

## 🎯 Objectives

- Analyze factors affecting appliance energy consumption  
- Build and compare multiple regression models  
- Predict energy usage accurately  
- Develop an interactive **Streamlit dashboard**  

---

## 📊 Dataset

- Dataset: **Appliances Energy Prediction Dataset**
- Features include:
  - 🌡️ Temperature (T1–T9)
  - 💧 Humidity (RH_1–RH_9)
  - ⏰ Time-based features (hour, day, month)
  - 🌬️ Weather conditions (pressure, windspeed, etc.)

## 🤖 Machine Learning Models

The following regression models were implemented and compared:

- Linear Regression  
- Ridge Regression  
- Decision Tree Regressor  
- Random Forest Regressor ⭐  
- Gradient Boosting Regressor  
- XGBoost Regressor ⭐  

---

## 📈 Performance Evaluation Metrics

- Mean Squared Error (MSE)  
- Root Mean Squared Error (RMSE)  
- Mean Absolute Error (MAE)  
- R² Score  

---

## 📊 Key Insights

- ⚡ Energy consumption peaks during **evening hours**
- 🌡️ Temperature significantly impacts energy usage  
- 💰 Shifting usage to off-peak hours reduces cost  
- 🌍 Machine learning enables **smart energy optimization**
